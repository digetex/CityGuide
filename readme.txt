Далее нужно создать в mysql пользователя с логином root и паролем 123456 (или настроить под себя в файле application.properties)
Так же в application.properties указать url к базе данных или сделать у себя такой же как файле
Для взаимодействия с с ботом нужно его найти в поиске по id @Digetex_bot. Наименование бота City Guide. Токин: 801730328:AAEaFPhjl-mW4u68APd2_KZerp7r_xb2r8I
